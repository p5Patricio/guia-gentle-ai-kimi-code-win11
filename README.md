# 🌙 Guía: Gentle-AI + Kimi Code en Windows 11 Nativo

> Guía gratuita y open-source para dominar el agente de coding con 256K de contexto.

## 📖 ¿Qué es esta guía?

Esta guía te enseña a instalar y configurar **Kimi Code CLI** (el agente de coding de Moonshot AI con modelo K2.6 y **256K tokens de contexto**) potenciado con el **ecosistema Gentle-AI**, todo de forma **nativa en Windows 11**.

Kimi Code destaca por su velocidad (~100 tokens/s) y su ventana de contexto enorme. Combinado con Gentle-AI, se transforma en un sistema de desarrollo completo con memoria persistente, workflow estructurado y documentación en tiempo real.

## 🎯 ¿Qué vas a conseguir?

| Feature | Descripción |
|---------|-------------|
| 🧠 **256K de contexto** | Comprende proyectos enteros sin perder el hilo |
| 💾 **Engram** | Memoria persistente entre sesiones via SQLite |
| ⚡ **SDD Workflow** | 9 fases: explore → propose → spec → design → apply → test → review → verify → archive |
| 📚 **Context7** | Documentación actualizada de librerías via MCP |
| 🔍 **GGA** | Guardian Angel — code review automático en cada commit |
| 🎯 **Modo Thinking** | Razonamiento profundo para decisiones complejas |
